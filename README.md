# StockMaster IMS

**StockMaster IMS** is a full-featured Inventory Management System for warehouses, combined with a modern **authentication system** with Signup, Login, Forgot Password, and OTP verification flows. Built with **React.js**, **Node.js**, and **MongoDB**, it provides seamless warehouse and stock management for admins and staff.

---

## Project Overview

StockMaster IMS helps manage warehouses, track stock, and handle internal operations. The authentication system ensures secure login/signup, password recovery, and OTP verification for user safety. The system has separate views for **Admins** and **Staff**.

---

## Features

### Authentication

- **Signup** – Register with name, email, and password  
- **Login** – Secure login with password visibility toggle  
- **Forgot Password** – Email-based OTP recovery  
- **OTP Verification** – Enter OTP to reset password  

### Warehouse & Stock Management

- **Warehouse Management** – Add, view, and manage warehouses  
- **Location Management** – Add locations/racks inside warehouses  
- **Stock Inventory** – Track stock levels, per-unit cost, and usage  
- **CRUD Operations** – Create, Read, Update, Delete items and locations  
- **Staff Operations** – Delivery Orders, Incoming Receipts, Internal Transfers, Stock Adjustments  

---

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Lucide Icons  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Routing**: React Router  
- **Authentication**: JWT / session-based (can be integrated)
